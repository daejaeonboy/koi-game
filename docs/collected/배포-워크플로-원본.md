description: 코이 연못 게임 배포 방법(수동 방식)
---

시스템에 Git이 설치되어 있지 않다면, 가장 쉬운 배포 방법은 **Netlify Drop**을 사용하는 것입니다.

## 🚀 넷리파이 드롭(권장)

이 방법은 별도 설치가 필요 없고 1분 이내로 끝낼 수 있습니다.

1.  **빌드 폴더 위치 확인**:
    -   파일 탐색기에서 `c:\Users\최동준\Desktop\koi 1.3\dist` 경로를 엽니다.
    -   이 폴더에 배포용으로 빌드된 게임 파일이 들어 있습니다.

2.  **Netlify에 업로드**:
    -   [app.netlify.com/drop](https://app.netlify.com/drop)으로 이동합니다.
    -   "Drag and drop your site output folder here" 영역에 **`dist` 폴더**를 드래그 앤 드롭합니다.
    -   몇 초만 기다리면 사이트가 바로 배포됩니다.

3.  **공유**:
    -   Netlify가 임시 URL(예: `peaceful-koi-123456.netlify.app`)을 발급해 줍니다.
    -   해당 링크를 다른 사람과 바로 공유할 수 있습니다.

---

## 대안: 버셀 명령줄 도구(고급)

Vercel 배포를 선호한다면, Git 없이도 CLI 도구를 설치해 배포할 수 있습니다.

1.  Vercel CLI 설치: `npm i -g vercel`
2.  배포 실행: `vercel`
3.  안내 프롬프트에 따라 진행
