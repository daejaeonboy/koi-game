# 🎏 Zen Koi Garden (젠 코이 가든)

나만의 평화로운 연못에서 잉어를 키우고, 교배하여 희귀한 품종을 발견해보세요.

## 🎮 게임 즐기는 법

이 게임은 **수집**과 **힐링**이 핵심입니다. 플레이어는 다음과 같은 방식으로 게임을 즐길 수 있습니다:

### 1. 잉어 육성 (Growth)
- **먹이 주기**: 화면을 클릭하거나 먹이 버튼을 눌러 잉어에게 밥을 줍니다.
- **성장**: 밥을 먹은 잉어는 `치어(Juvenile)`에서 `성어(Adult)`로 성장합니다.
- **프리미엄 옥수수**: 상점에서 '옥수수'를 구매해 먹이면 3배 더 빨리 성장합니다! 🌽

### 2. 교배와 유전 (Breeding & Genetics) 🧬
- 성어가 된 잉어 두 마리를 선택하여 교배할 수 있습니다.
- **멘델의 유전 법칙**을 기반으로 부모의 색상과 패턴을 물려받습니다.
- **희귀 변이 발견**:
  - **기본 색상**: 갈색(Brown)에서 시작하여 적색, 백색, 흑색 등 다양한 색상 발견
  - **특수 모프**: 알비노(Albino), 황금(Gold), 투명(Transparent) 등 희귀한 돌연변이 도전!
  - **패턴**: 다양한 색상의 점(Spot) 패턴 유전

### 3. 경제 활동 (Economy) 💰
- 다 자란 잉어를 판매하여 **ZP (Zen Points)**를 획득하세요.
- 희귀한 색상과 패턴일수록 더 비싼 가격에 팔립니다.
- 모은 ZP로 먹이를 사고, 새로운 테마를 잠금 해제하세요.

### 4. 나만의 연못 꾸미기 (Customization) 🎨
- **테마 변경**: 모래, 자갈, 진흙, 이끼 등 다양한 바닥 테마를 적용해 분위기를 바꿔보세요.
- **감상 모드**: UI를 끄고 평화롭게 헤엄치는 잉어들을 바라보며 힐링하세요.

## 🛠️ 설치 및 실행 방법

1. 의존성 설치:
   ```bash
   npm install
   ```

2. 게임 실행:
   ```bash
   npm run dev
   ```

3. 빌드 및 배포:
   ```bash
   npm run build
   ```
