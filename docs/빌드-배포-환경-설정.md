# 빌드/배포/환경 설정

## 실행 스크립트
루트(package.json):
- npm run dev (Vite dev server, port 4000)
- npm run build
- npm run preview

Functions(functions/package.json):
- npm run lint
- npm run build
- npm run serve (에뮬레이터)
- npm run deploy

## 환경 변수
클라이언트 Firebase(services/firebase.ts):
- VITE_FIREBASE_API_KEY
- VITE_FIREBASE_AUTH_DOMAIN
- VITE_FIREBASE_PROJECT_ID
- VITE_FIREBASE_STORAGE_BUCKET
- VITE_FIREBASE_MESSAGING_SENDER_ID
- VITE_FIREBASE_APP_ID
- VITE_FIREBASE_MEASUREMENT_ID

기타:
- GEMINI_API_KEY (vite.config.ts에서 process.env.*로 주입)

## 프로그레시브 웹 앱
- index.html에서 Service Worker 등록
- public/sw.js: Network First + 캐시 폴백
- public/manifest.json: standalone PWA 메타

## 파이어베이스 배포 설정
- Hosting: dist 폴더 배포, SPA rewrite ** -> /index.html
- Functions predeploy: lint + build
- Firestore: rules/indexes 파일 연동

## 버셀 설정
- vercel.json에 SPA rewrite 포함
