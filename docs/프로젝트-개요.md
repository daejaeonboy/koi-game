# 프로젝트 개요

## 한 줄 요약
Zen Koi Garden은 React + Vite + Firebase 기반의 코이 육성/교배/장터 게임입니다.

## 핵심 기능
- 코이 육성: 먹이, 성장 단계(fry/juvenile/adult), 체력/수질/질병 시스템
- 유전/교배: 멘델 기반 색상/점 패턴, 점 표현형 유전자(채도) 시스템
- 경제: ZP(인게임 재화), AP(광고 포인트), 아이템/연못 확장/트로피 구매
- 온라인: Google 로그인, 클라우드 저장, 세션 충돌 감지(중복 접속 방지)
- 장터: 판매 등록, 입찰, 즉시구매, 취소, 만료 처리
- 업적/랭킹: 업적 해금/보상, 명예 점수/업적 점수 랭킹
- 멀티플랫폼: 웹(PWA)

## 기술 스택
- 프론트엔드: React 19, TypeScript, Vite 6, Tailwind(CDN)
- 백엔드: Firebase Auth/Firestore/Functions/Hosting
- 배포 보조: Firebase Hosting, Vercel 설정 파일 공존

## 엔트리 포인트
- 웹 진입점: index.tsx
- 메인 앱: App.tsx
- 인증 컨텍스트: contexts/AuthContext.tsx
- 게임 렌더/물리: components/Pond.tsx + utils/GameEngine.ts + utils/koiRenderer.ts
- 클라우드 함수: functions/src/index.ts
